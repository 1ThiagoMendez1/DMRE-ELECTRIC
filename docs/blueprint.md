# **App Name**: VoltAI Engineering

## Core Features:

- Animated Hero Section: A landing section featuring animated electric neurons and a futuristic title and call to action.
- Dynamic Project Collage: A dynamic grid displaying images of electrical projects with hover effects and project descriptions.
- Admin Project Management: A backend CRUD system to manage the project collage including image uploads, descriptions, and categories.
- Contact Form: A modern contact form with animated fields and a stylized map.
- Admin User Authentication: A secure authentication system with JWT for admin users to manage the website content.
- AI-Enhanced Visuals: Generative AI algorithms drive animations which adapt according to server load. Reasoning tool decides whether to increase animations during idle periods.

## Style Guidelines:

- Primary color: Electric Blue (#7DF9FF) for a futuristic and energetic feel.
- Background color: Matte Black (#222222) to provide contrast and depth.
- Accent color: Neon Green (#39FF14) for highlights and interactive elements.
- Body and headline font: 'Space Grotesk', sans-serif, for a techy, scientific look.
- Use icons with energetic effects that reflect electrical themes. Each icon should appear to glow.
- Implement a dynamic grid layout for the project collage, ensuring responsiveness across all devices. Organize sections logically to maintain flow.
- Incorporate subtle animations with GSAP or Framer Motion to enhance user engagement and visual appeal, focusing on electric pulses and waves.
